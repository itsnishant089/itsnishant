## 🚀 Setup & Run

**Step 1 — Run Locally**

Must run via a local server (not `file://` directly):
```bash
# Python
python -m http.server 5500
# Or VS Code → Right-click index.html → Open with Live Server
```

**Step 2 — Profile Image**

Replace `image/experience.jpeg` with your photo. Recommended: `300×300px` square.

**Step 3 — EmailJS (Contact Form)**

Already configured in `js/main.js`:
```js
emailjs.init({ publicKey: 'tSLSXcGfuU3S3XDr4' });
emailjs.sendForm('service_7dunjfz', 'template_4sg47r9', contactForm);
```

**Step 4 — Deploy to Vercel**
```bash
npm i -g vercel && vercel
```
Or connect the GitHub repo on [vercel.com](https://vercel.com) — auto-detects static HTML.
